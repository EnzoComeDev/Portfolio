# Nuit de l'informatique 2025 - Défi "Page Ergonomie"

Ce projet a été réalisé dans le cadre de la **Nuit de l'informatique 2025**.  
L'objectif du défi **"Ergonomie"** était de créer une page web **tout sauf ergonomique**, pour comprendre les enjeux de conception et tester les limites de l'expérience utilisateur.

---

## Description

La page contient un formulaire de connexion avec un **mot de passe complexe**. L'utilisateur doit respecter une série de contraintes pour réussir à soumettre le formulaire. Les contraintes sont volontairement complexes et décalées pour rendre l'expérience frustrante et peu ergonomique


Le bouton **Submit** se déplace aléatoirement lorsque les conditions ne sont pas toutes remplies, empêchant l'utilisateur de cliquer facilement.

---

## Technologies utilisées

- **HTML / CSS / JavaScript**  
- Manipulation du DOM et gestion des événements pour créer un formulaire interactif et “réactif”  
- Logique complexe pour la validation du mot de passe et le déplacement dynamique du bouton  

---

## Objectif pédagogique

- Explorer les limites de l'ergonomie et du design d'interface utilisateur  
- Pratiquer la manipulation DOM avancée avec JavaScript  
- Créer une expérience volontairement déroutante pour comprendre l'importance d'une interface claire et intuitive
